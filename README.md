## 學 TypeScript 源起
在開發 app 的這幾個月下來，慢慢感受到專案愈來愈大時，除錯功力也需要跟著提升。
但是有些 bug ，真的不好解決，如果可以在專案建置開始，則透過好的方法(或工具)來建置，
當專案愈來愈大時，也會相對容易除錯及維護。

聽到許多團隊使用 TypeScript ，稍微了解後，知道這大概是我可以考慮使用的工具，
便開始了這趟學習之旅。
